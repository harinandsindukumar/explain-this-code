# Explain This Code

A beginner-friendly open-source tool that explains code in simple language.

## Author
Harinand Sindukumar (harinandsindukumarkg@gmail.com)

## Who is this for?
- Students
- Beginners
- Self-learners
- Teachers

## Features
- Plain English explanations
- Line-by-line breakdown
- Common beginner mistakes
- No login required

## How to run
1. Start backend
   cd backend
   npm install
   node server.js

2. Open frontend/index.html in browser

## Ethics
- No code is stored
- Explanations may be imperfect
- Meant for learning, not shortcuts
